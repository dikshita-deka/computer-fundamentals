# Hardware Vs Software

## What is Hardware?

Hardware refers to the physical parts of a computer that we can see and touch.
These are the actual components that make up a computer system.

Examples:
- CPU
- RAM
- SSD
- Monitor
- Keyboard
- Mouse

In simple words, hardware is everything that we can physically pick up or plug into a computer.

---

## What is Software?

Software refers to the programs, applications, instructions that tell the hardware what to do.
Software gives life to the hardware by providing instructions.

Examples:
- Windows
- Linux
- Microsoft Excel
- Whatsapp
- Games
- Applications

Software can not be touched. It exists in the form of code and instructions.

---

## Key Differences

Hardware:
- Physical Components
- Can be touched
- Manufactured in Fatories
- Can be physically damaged

Software:
- Programs and Instructions
- Can not be touched
- Developed and coded by programmers
- Cannot be physically damaged, but can be corrupted or infected.

---

## Relationship between Hardware and Software

Hardware and Software are complementary to each other.
Hardware without software is useless because it cannot execute instructions.
Software without hardware cannot run because it needs a physical system to operate.

In simple terms:
Hardware provides the structure.
Software provides the instructions.

Both depends on each other to function properly.

---

## Real-Life Analogy
Hardware is like the human body.
Software is like the brain.

Without the brain, the body cannot perform actions.
Without the body, the brain cannot function.

Both are necessary.

---

## Cybersecurity Perspective
Both hardware and software can be attacked.

- Malware and viruses mainly target software.
- Hardware components can also be targeted (eg: Firmware attacks)

 Security is important at both levels.

 Understanding hardware and software helps in understanding where vulnerabilities can exist and how systems can be protected.
